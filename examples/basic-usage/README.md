# Basic Usage Example

Demonstrates automatic reflection (`AutoReflectable`), `from_value` / `to_value`, `std::optional`, `std::vector`, nested structs, and roundtrip with a minimal `ValueLike` implementation.

## Run

```sh
cd examples/basic-usage
mise exec -- exon sync
mise exec -- exon build
mise exec -- exon run
```

See the root README for more context on txn.
